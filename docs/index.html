Colors

The classes are named using the format color-{name}[-bg][-{shade}]

  • name - refer to the table below for valid colors
  • bg (optional) - to color the background leave out if coloring text
  • l or d (optional) - to shade lighter for darker

Examples: color-white, color-white-bg, color-white-bg-d

Spacing

The classes are named using the format {property}{side}-{size}.

Where property is one of:

  • m - for classes that set margin
  • p - for classes that set padding

Where sides is one of:

  • blank - for all 4 sides
  • t - top
  • b - bottom
  • l - left
  • r - right

Where size is one of:

  • 0 - eliminate margin or padding
  • 1 - 1rem
  • 2 - 2rem
  • 4 - 4rem
  • 6 - 6rem
  • 8 - 8rem

Examples: pb-0, mt-1, etc.
